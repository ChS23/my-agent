## Создание нового скилла

Пользователь хочет создать новый скилл. Помоги ему:

1. **Уточни детали**: что скилл должен делать, когда активироваться (auto/manual), какие инструменты использовать
2. **Сгенерируй файл** в формате YAML frontmatter + markdown инструкции
3. **Сохрани** результат — покажи пользователю готовый файл

### Формат скилла

```
---
name: skill_name
description: Краткое описание — когда использовать
trigger: manual или auto
enabled: true
---

Инструкции для агента в свободной форме.
Можно использовать любые доступные tools.
```

### Правила
- name: только латиница, цифры, подчёркивания
- description: конкретное, чтобы агент понимал когда активировать (для auto)
- trigger: `auto` — агент сам активирует по контексту, `manual` — только по /команде
- Инструкции пиши на том языке, на котором общается пользователь
- Используй доступные tools: memory_search, memory_store, web_search, ticktick_list, read_url и др.
- Формат ответа агента: Telegram HTML

### После генерации
Покажи пользователю готовый файл и скажи, что нужно:
1. Сохранить в `skills/имя.md`
2. Перезапустить агента

Добавь кнопку "📁 Сохранить скилл" (callback: сохрани скилл).
